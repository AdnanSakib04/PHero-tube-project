Discuss the scope of var, let, and const

var: var is functional scope. Which means if we declare a variable using var we can access it throughout the function and we can also access it outside the function. If we globally declare a variable using var then it can be accessed from anywhere. We can redeclare a variable using var and update its value.

let: let is block scoped. Which means if we declare a variable using let inside a block, we can only access it within its block. If we declare it globally then we can access it from anywhere. We can update the value of any variable declared using let within the same block, but we can not redeclare within the same block.

const: const is block scoped. It can only be accessed within a block if we declare it inside a block. If we declare it globally then we can access it from anywhere. Within a block we cannot redeclare using const, also we cannot update its value.
